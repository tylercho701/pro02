개발 언어 및 시스템
 - 자바 버전 : JDK 1.8
 - 자바 스크립트 버전 : 1.0
 - 다이나믹 랩 모듈 : 3.1
 - DBMS : Oracle 11g xe, Oracle SQL Developer
 - 자바 라이브러리 : crypto, json, cos, commons, mail
 - FrameWork : BootStrap 3.3.2
 - 스크립트 플러그인 : jQuery 1 .12 .4
 - 스크립트 api : daum 주소 api, json 데이터 처리, iamport 결제 api
 - 프로젝트 데이터 처리 패턴 : MVC2 (View -> Controller -> Model -> Controller -> View)

개발 언어의 트렌드
 2022년 티오베지수 뿐만 아니라 PYPL 순위에서도 상위권에 랭크되고 웹 기반 시스템의 대부분에서 활용되고 있으며, 재사용성과 보안성이 높은 'JAVA' 기반의 웹 애플리케이션을 개발할 것입니다.
MVC2 패턴의 처리 방식을 채택하여 보안성 강화를 위한 jsp와 el, jstl, 암호화 기능을 활용할 것이며, model, view, controller로 처리 기능을 분할하여 jsp를 개선하였습니다.
